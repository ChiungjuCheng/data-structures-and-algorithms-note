# Intervals
常見題目類型，給定一個二維陣列，每個陣列內包含兩個數字如下圖:
```
[[0,1] , [3,5] , [7,10]]
```
會有各種題型，要求新的Intervals不要overlapping。
```
// overlapping的Intervals
[[0,1] , [3,5], [4,10]]
```
